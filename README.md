# Pipeline Operations & Integrity Dashboard

## 📌 Project Overview
Developed an automated monitoring solution for energy infrastructure to visualize real-time pipeline health. This project bridges the gap between raw operational sensor logs and executive decision-making.

## 🛠️ Tech Stack
* **Data Source:** CSV / Excel
* **Analysis Tool:** Power BI
* **Logic:** DAX (Data Analysis Expressions)
* **Visualization:** Geospatial Mapping & Conditional Formatting

## 🚀 Key Features
* **Live Status Map:** Uses Latitude/Longitude coordinates to plot station health.
* **Conditional Alerts:** Automatically flags "Critical" or "Warning" status using red/yellow indicators.
* **Cross-Filtering:** Interactive bar charts that filter based on map selection.
* **Operational Metrics:** Calculated Average Pressure and Flow Rates using custom DAX measures.
